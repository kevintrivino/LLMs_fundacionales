# LLMs_fundacionales
Se testean 3 LLMs fundacionales (Robertuito, bert multilingual, beto) con frases cualquiera que simulan comentarios en redes sociales, para evaluar la lógica de cada modelo.

## Objetivo
Funciona como primer acercamiento a modelos de lenguaje para clasificación de sentimiento. La clasificación se hace con las etiquetas 'POS','NEU','NEG'. A partir de este ejercicio, se puede entender por qué surge la necesidad de hacer fine-tunning en modelos de lenguaje para capturar con precisión la intención del texto dada una jerga propia en cada región o el sarcasmo, los cuales en ocasiones pueden cambiar por completo el significado de un comentario.
